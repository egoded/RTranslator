# Инструкции для AI-ассистента

## Язык общения
- Всегда отвечай на **русском языке**
- Комментарии в коде пиши на русском
- Коммит-сообщения пиши на русском

## Проект
- Это форк проекта [RTranslator](https://github.com/niedev/RTranslator) — офлайн-переводчик в реальном времени для Android
- Язык: Java, система сборки: Gradle
- Основной код: `app/src/main/java/nie/translator/rtranslator/`
- Минимальные требования: 6 ГБ RAM на устройстве

## Стек и зависимости
- AI-модели: NLLB (перевод), Whisper (распознавание речи) — работают через OnnxRuntime
- Bluetooth: библиотека BluetoothCommunicator
- Токенизация: SentencePiece
- Определение языка: ML Kit (Google)

## Правила работы с кодом
- Не ломай существующий функционал при доработках
- Перед изменением файла — сначала прочитай его
- Не добавляй лишних зависимостей без явной необходимости
- Тестируй изменения перед коммитом
